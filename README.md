# buffer-mobile

A Flutter clone of [Buffer](https://gitlab.gnome.org/cheywood/buffer) — the minimal ephemeral
text editor for GNOME — targeting Android and iOS.

The app opens to a blank page. The user types. The text is gone when the app closes.
The last 10 notes are silently preserved and recoverable.
